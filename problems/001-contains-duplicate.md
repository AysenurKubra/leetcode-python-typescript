# 001 - Contains Duplicate

Given an integer array nums, return true if any value appears more than once in the array, otherwise return false.

## Examples

- Input: nums = [1, 2, 3, 3]  
  Output: true

- Input: nums = [1, 2, 3, 4]  
  Output: false

## Recommended Complexity

- Time: O(n)  
- Space: O(n)
